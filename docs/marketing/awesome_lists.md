## Entry variations by context

**For AI/Agent lists:**
> AI agent framework with 22 built-in tools, three-tier memory, and 97% tool accuracy. Supports Anthropic, OpenAI, Ollama, Gemini. 7.5 MB single binary.

**For Rust lists:**
> Async AI agent built with tokio + axum + teloxide + serenity. 6K lines, zero clippy warnings. SQLite-backed memory with vector search and knowledge graph.

**For self-hosted lists:**
> Self-hosted AI agent — single 7.5 MB binary, 14 MB RAM. Telegram + Discord bots, system monitoring, email scanning, GitHub integration. Runs on a Raspberry Pi.

**For SQLite lists:**
> Uses SQLite as sole backend for: conversation history, vector memory (embedding BLOBs + cosine similarity), and knowledge graph (entity-relation with soft-delete). Zero external services.

**For MCP lists:**
> MCP client implementation in Rust. Connects to any MCP server via stdio JSON-RPC. Tools auto-discovered at startup, transparently merged with 22 built-in tools.

**For Telegram/Discord lists:**
> AI-powered Telegram/Discord bot with tool calling, long-term memory, server management, GitHub integration, and system monitoring. 7.5 MB, runs anywhere.
